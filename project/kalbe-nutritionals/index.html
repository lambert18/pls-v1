About This Project

This project is to meet the company's need to read core data through graphs. The core data must be cleaned and analyzed first to ensure whether the required data can be made perfect or not, because the data may not be complete. Python is built-in and with the Plotly Framework & Library as chart requirements, Dash is for creating interactive web applications, and the dash bootstrap components are used for integration between the bootstrap framework and the plotly dash.
